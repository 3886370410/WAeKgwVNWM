# 个人博客管理系统 java1180

## 项目概述


## 技术栈

- **后端：**
  - Java
  - Spring
  - Springboot
  - MVC模式
  - Mybatis
  - MySQL
  - Maven
- **前端：**
  - HTML
  - JavaScript
  - CSS
  - Vue.js
- **开发工具：**
  - IntelliJ IDEA
  - VS Code

## 功能模块

1. 用户管理
2. 博客发布与编辑
3. 评论管理
4. 分类与标签管理
5. 数据统计与分析

## 系统角色

- 博主：发布和管理博客内容
- 访客：浏览博客、发表评论

## 运行环境

- MySQL数据库
- JDK 1.8
- Redis
- Node.js 17+

## 部署步骤

1. 环境搭建：
   - 安装MySQL数据库
   - 安装JDK 1.8
   - 安装Redis
   - 安装Node.js 17+
2. 后端部署：
   - 使用IntelliJ IDEA导入项目
   - 配置数据库连接
   - 编译并运行项目
3. 前端部署：
   - 使用VS Code打开前端项目
   - 安装依赖
   - 编译并运行项目
